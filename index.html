Le module PushalotRemote permet d'utiliser le service Pushalot pour envoyer des notifications sur les platformes Windows 8 et Windows Phone.

Ce service est aussi supporté par IFTTT pour envoyer des notifications aux smartphones/tablettes Windows via cette page de configuration

Le paramètre push permet de véhiculer le message à envoyer. Le titre du message est définit dans les propriétés.

Pour configurer le modules vous devez:

  • Installer l'application Windows 8 ou Windows Phone.
  • Créer un compte et application pushover.
  • Renseigner dans SARAH l'identifiant (Token) du compte et les paramêtres de l'application.

Réalisé sur les bases du plugin Pushover de JP Encausse
